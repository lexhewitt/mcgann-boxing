# Fleetwood Boxing Gym - Management System

A comprehensive booking and management application for Fleetwood Boxing Gym, built with React, TypeScript, Supabase, and Stripe.

## Quick Start

**Prerequisites:** Node.js (v20+), Supabase account, Stripe account

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Set up environment variables:**
   - Create `.env.local` file in the root directory
   - Add your Supabase and Stripe credentials (see [CONFIG.md](CONFIG.md) for details)

3. **Run the app:**
   ```bash
   npm run dev
   ```
